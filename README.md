# sprotyvmap

**SprotyvMap** це вебдодаток для пошуку найближчих ТЦК.

## Технології:
- .NET 8 / C#
- Blazor WASM

## Сервіси:
- Firebase (Хостинг вебсайту)
- Azure (Хостинг API)
- GitHub (CI/CD)
- Visicom Data API (Геокодування)
